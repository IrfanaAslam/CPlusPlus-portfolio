# Student Record System

A console application that tracks students, computes GPA, ranks students, and exports records to CSV.

## Features
- Add students and grades
- Compute GPA per student
- Rank students by GPA using `std::sort` with a custom comparator
- Export records to CSV

## Build & Run
```bash
g++ main.cpp -o student_record_system
./student_record_system
```
